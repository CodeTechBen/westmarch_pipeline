# Westmarch Pipeline

## Introduction
This pipeline will be created to better manage you DND beyond audience in westmarches, in order to better manage the items being given out and enable other usecases that are brought on by managing many simultaneous ad-hoc parties.

## Database
We utilise a postgres database, you must have *postgres* installed and then you can run source .database/create_db.sh this will create the database, you can then populate the tables using source .database/reset_schema.sh
